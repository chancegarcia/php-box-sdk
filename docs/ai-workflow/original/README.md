# Original Templates

This folder holds unmodified copies of the template files as received from the
template source. It is **empty in the source repo** — populate it in each project
that copies the templates.

## How to Use

Before customizing any template file, copy it here first:

```bash
cp single-repository-workflow.md original/
```

Then edit the main file freely. The original stays pristine as your baseline.

## What the Originals Are For

- **Diff baseline** — see exactly what your project changed vs. what it received
- **Merge aid** — when a new template version is available, three-way merge:
  original → new template → your customized version
- **Contribution target** — if a project-specific improvement is worth backporting,
  update the original here and note the source project
